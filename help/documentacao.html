Visão Geral

Este sistema permite a inserção de expressões regulares para gerar uma tabela de análise léxica, visualizar o autômato correspondente e ver a análise léxica em ação.

Funcionalidades

  • Inserir Expressão Regular
  • Gerar Tabela de Análise Léxica
  • Mostrar Autômato
  • Executar Análise Léxica

Como Usar o Sistema

  1. No campo "Insira sua Expressão Regular", digite a expressão regular desejada.
  2. Clique no botão "Gerar Tabela de Análise Léxica" para gerar a tabela correspondente.
  3. Clique no botão "Mostrar Autômato" para visualizar o autômato gerado pela expressão regular.
  4. Clique no botão "Executar Análise Léxica" para ver a análise léxica funcionando com a expressão inserida.

Exemplo de Expressão Regular

[a-zA-Z_][a-zA-Z0-9_]*

Esta expressão regular corresponde a identificadores válidos em muitas linguagens de programação, que começam com uma letra ou sublinhado, seguidos de letras, números ou sublinhados.

Descrição dos Campos

  • Expressão Regular: Campo para inserir a expressão regular que será utilizada na análise léxica.
  • Gerar Tabela de Análise Léxica: Botão que gera a tabela de análise léxica a partir da expressão regular fornecida.
  • Mostrar Autômato: Botão que exibe o autômato gerado pela expressão regular.
  • Executar Análise Léxica: Botão que executa a análise léxica com base na expressão regular inserida.
